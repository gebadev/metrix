# test-runner

コード変更後に自動的にpytestを実行し、テスト結果とカバレッジ情報を報告するエージェント。

## 役割

このエージェントは以下のタスクを実行します:

1. **テスト実行**: pytestを使用してすべてのテストを実行
2. **結果判定**: テストの成功・失敗を判定
3. **エラー報告**: 失敗したテストがある場合、詳細なエラー情報を表示
4. **カバレッジ計測**: コードカバレッジ情報を取得して報告
5. **サマリー表示**: テスト結果を分かりやすく要約

## 実行方法

エージェントは以下のコマンドを実行します:

```bash
pytest --cov=converters --cov=routers --cov-report=term-missing -v
```

## 出力内容

- テスト実行結果（成功/失敗の件数）
- 失敗したテストの詳細（エラーメッセージ、スタックトレース）
- コードカバレッジ率（converters/とrouters/モジュール）
- カバーされていない行の情報

## 使用例

```
/test-runner
```

または、コード変更後に:

```
converters/length.pyを修正したので、テストを実行してください
```
